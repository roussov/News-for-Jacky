
# News for Jacky!
Extraire les données d'un site  quelconque et d'envoyer le résultat sous forme de courrier au destinataire de son choix...


## Installation

Utilisez le terminal et lancez les commandes:

```bash
chmod +x start.sh
./start.sh
```
